# Glosario

| Término | Definición |
|---------|-----------|
| Tour | Producto turístico individual con precio, duración, imágenes, incluye/no incluye, itinerario |
| Categoría | Clasificación del tour: aventura, cultural, playa, gastronomico, naturaleza |
| Itinerario | Lista ordenada de actividades con hora y descripción |
| CTA | Call to action — botón que invita a una acción principal |
| Hero | Sección principal de landing con imagen de fondo, título y CTA |
| Card | Componente de presentación de tour con imagen, nombre, precio, duración |
| Tripulación | [PENDIENTE: definir si se usa este término] |
| RSC | React Server Component |
| SSG | Static Site Generation — páginas pre-renderizadas en build |
